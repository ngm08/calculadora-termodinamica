Calculadora de Propriedades Termodinâmicas
Este projeto foi desenvolvido para facilitar o acesso a dados de entalpia e entropia

Fórmulas utilizadas:
Utilizamos a equação de Van der Waals:
$$(p + \frac{a}{v^2})(v - b) = RT$$
